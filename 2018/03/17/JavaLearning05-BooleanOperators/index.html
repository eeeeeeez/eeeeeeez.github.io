JavaLearning05-BooleanOperators

Posted on 2018-03-17 |

Boolean Operators

  • ####and &&
    It returns a boolean value of true only when the both sides of && are true. 

    System.out.println(true && true);
//prints true

System.out.println(false && true);
//prints false

System.out.println(true && false);
//prints false

System.out.println(2 < 3 && 4 < 5);
//prints true, because the both statements are true.

  • ####or ||
    It returns a boolean value of true when at least one expression on either side of || is true.

    System.out.println(false || false);
//prints false

System.out.println(false || true);
//prints true

System.out.println(2 > 1 || 3 > 4);
//prints true, because the left statement is right.

  • ####not !
    It returns the opposite of the expression immediately after it. It will return false if the expression is true, and true if the expression is false.

    System.out.println(!false); 
//not false, then print true

System.out.println(!true);
//not true, then print false

System.out.println( !(4 <= 10) ) ;
// expression is true, not true then print false

Precedence of boolean operators

  1. ! is evaluated first
  2. && is second

  3. || is third

    System.out.println( !(false) || true && false); 
//!(false) -> true
//true && false -> false
//true || false -> true
//therefore, print true
